## Prerequisites

* Linux, Macos, WSL
* PHP >=8.2
* make >=4.3
* npm >=10.8
* node >=20.1


## Setup

```bash
git clone https://github.com/SergDanilov/php-project-57.git
cd php-project-57
make setup
```
## Usage

```bash
make start
```
* You can see app on localhost, port 8000. http://localhost:8000/
