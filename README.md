## Prerequisites

* Linux, Macos, WSL
* PHP >=8.2
* Make
* Git


## Setup

```bash
git clone https://github.com/SergDanilov/php-project-57.git
cd php-project-57
make setup
```
## Usage

```bash
make start
```
